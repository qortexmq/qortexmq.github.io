üöÄ Quick Start

Start the server and use .NET client to send and receive messages:

dotnet@localhost:~
$ docker run -d --name qortexmq-server -p 9090:9090 -p 9080:9080 qortexmq-server
$ # Open http://localhost:9080 for web interface

Now use .NET client to send messages:

Publisher.cs
using QortexMQ;
using QortexMQ.Common.Enums;
// Create client with modern API
using var client = new QortexMqClient("127.0.0.1", 9090);
await client.ConnectAsync();
// Send message with high priority
"await client.PublishAsync<string>("orders", "New order #123", MessagePriority.High);"

And subscribe to topic to receive messages:

Subscriber.cs
// Subscribe to topic with handler
await client.SubscribeAsync<string>("orders", async (message) =>
{
Console.WriteLine($"Received order: {message}");
await Task.Delay(100); // Processing
}, maxHandlers: 1); // 1 parallel handler
