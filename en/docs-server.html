üì• QortexMQ Server Installation

Installation Methods

1. Native Distributions (Recommended)

Download ready executable file for your platform:

  • Windows: QortexMQ.Server-v1.3.0-x64.zip
  • Linux: QortexMQ.Server-v1.3.0-linux.zip
  • macOS: QortexMQ.Server-v1.3.0-macos.zip

Step-by-step Instructions:

  1. Go to the link: distributions on Downloads page
  2. Download the archive for your operating system
  3. Extract the archive to a convenient folder
  4. Start the server:
Windows
C:\> QortexMQ.Server.exe
Linux/macOS
$ chmod +x QortexMQ.Server
$ ./QortexMQ.Server

2. Docker Image (Alternative Method)

Prerequisites:

  • Docker Desktop or Docker Engine installed
  • Internet connection for downloading the image

Step-by-step Instructions:

  1. Go to the link: image on Downloads page
  2. Download the file: qortexmq-server.tar.gz
  3. Load into Docker:
docker@localhost:~
$ docker load < qortexmq-server.tar.gz

3. Pull from Docker Hub (recommended for cloud usage)

Prerequisites:

  • Docker Desktop or Docker Engine installed
  • Internet connection to download the image
  • Docker Hub account (optional, for private images)

Step-by-step instructions:

  1. Pull the image from Docker Hub:
docker@localhost:~
$ # Pull latest version
$ docker pull qortexmq/qortexmq-server:latest
$
$ # Or pull specific version
$ docker pull qortexmq/qortexmq-server:1.3.0

Note: Use the official repository qortexmq/qortexmq-server or replace with your username if you've uploaded the image to your repository.

Running the image

After pulling the image from Docker Hub, run it with a single command:

docker@localhost:~
$ docker run -d \
$ --name qortexmq-server \
$ -p 9090:9090 \
$ -p 9080:9080 \
$ qortexmq/qortexmq-server:latest

üöÄ Quick Start

Simple Startup (Recommended)

docker@localhost:~
$ docker run -d \
$ --name qortexmq-server \
$ -p 9090:9090 \
$ -p 9080:9080 \
$ qortexmq-server:latest

Running with Environment Variables

docker@localhost:~
$ docker run -d \
$ --name qortexmq-server \
$ -p 9090:9090 \
$ -p 9080:9080 \
$ -e QMQ_WebServer__UseWebServer=true \
$ -e QMQ_WebServer__Urls=http://0.0.0.0:9080 \
$ -e QMQ_Server__Ip=0.0.0.0 \
$ -e QMQ_Server__Port=9090 \
$ qortexmq-server:latest

üîç Problem Solving

Problem: Port Already in Use

docker@localhost:~
$ # Find process using the port
$ lsof -i :9090
$ lsof -i :9080
$
$ # Stop container
$ docker stop qortexmq-server

Problem: Container Won't Start

docker@localhost:~
$ # Check logs
$ docker logs qortexmq-server
$
$ # Check status
$ docker ps -a | grep qortexmq

Problem: Image Not Found

docker@localhost:~
$ # Check available images
$ docker images | grep qortexmq
$
$ # If image is missing, build from source code
$ git clone https://github.com/your-username/QortexMQ.git
$ cd QortexMQ/src
$ docker build -f QortexMQ.Server/Dockerfile -t qortexmq-server:latest .
