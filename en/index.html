FAQ

  • What features does QortexMQ.Client provide? Dependency Injection, Builder pattern, automatic reconnection, configurable timeouts, TCP optimizations, structured logging.
  • What delivery modes are supported? Single (to one subscriber) and Broadcast (to all subscribers). Message priorities: High and Low.
  • Is there a web interface for monitoring? Yes, built-in web interface with real-time statistics of topics, subscribers and messages.
  • Is ASP.NET Core supported? Yes, full compatibility with ASP.NET Core, Worker Services, Background Services and Dependency Injection.
